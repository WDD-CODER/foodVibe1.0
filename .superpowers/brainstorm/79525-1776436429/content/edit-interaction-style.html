A — Click any field to edit it

Fields look like read-only labels by default. Clicking one turns it into an input. A pencil icon hints that the row is editable. Active row stays in edit mode until you click elsewhere.

B — Always-edit form

The preview shows up as a mini-form from the start — all fields are inputs, no click required. Feels more like a lightweight builder, less like a read-only preview. Simpler to implement.

C — Section-level edit toggle

Each section (header info, ingredients, steps) has its own "Edit" button. Clicking it flips that section into edit mode. Other sections stay read-only until toggled. More structured, less accidental edits.
